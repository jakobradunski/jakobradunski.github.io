Random Color Generator

Check out the live demo and the source code!

Live Demo Source Code

"Random Color Generator" is a website that lets you generate random colors with a focus on a seamless user experience by
  • utilizing Web APIs like the Web Storage API for keeping the same colors upon a site refresh, and
  • the Clipboard API for copying colors to your clipboard!

The weppage is also responsive, and the text adapts too!

There are lots of optimizations that make the site pretty fun, so feel free to check it out!
